## 📌 Project Overview
This project presents a complete **Annual Sales Report Analysis Dashboard** created using **Tableau** for the brand **Seneohl**.  
The dashboard helps analyze sales performance, customer behavior, GST impact, and order trends across different states, months, and order statuses.

It transforms raw business data into interactive visual insights for faster and smarter business decision-making. Like turning spreadsheets into a command center 🚀

---

# 🎯 Objectives

- Analyze total sales performance
- Identify high-performing states
- Track monthly sales trends
- Compare order statuses
- Monitor GST payments
- Understand average order values
- Improve business decision-making using data visualization

# 📈 Dashboard Features

## 1. Total Sales by States
- Shows sales distribution across Indian states
- Helps identify top-performing regions

## 2. Total Sales by Months
- Monthly sales trend analysis
- Detects seasonal spikes and low-performance months

## 3. Average Value by Order Status
- Compares average invoice value for:
  - Delivered
  - Returned
  - Cancelled
  - Shipped
  - Ordered

## 4. KPI Cards
The dashboard includes major business KPIs:

| KPI | Value |
|-----|------|
| Total Sales | 260,978 |
| Total Orders | 1,588 |
| Highest Order Price | 460.2 |
| Average Order Price | 158.2 |
| Total GST Paid | 9,381 |

## 5. Sales by Status
- Displays total taxable sales by order status
- Helps evaluate operational efficiency

## 6. Interactive Filters
Users can filter data using:
- Status
- GST Rate
- HSN Code
- Month
- State

---

# 📊 Key Insights

- Bihar generated one of the highest sales volumes
- Delivered orders contributed the highest revenue
- April showed peak monthly sales
- Average order value remained consistent across statuses
- GST contribution crossed ₹9,000
